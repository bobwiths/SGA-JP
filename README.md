# SGA-JP
## Sistema de Gestão de Alunos - Projeto do Programa Jovem Programador

Estrutura do Projeto:

> Esta será a documentação do meu projeto

## Menu de seleção de módulos do sistema
### O SGA deverá contar com um módulo de escolha de:
1. Cadastro de novos alunos
2. Listagem dos alunos cadastrados
3. Alteração cadastral dos alunos
4. Exclusão de alunos
> 0 - Para sair do sistema

## Cadastro de novos alunos
### O sistema deverá realizar o cadastro de alunos, onde incluirá:
- Cadastro de:
  - Nome
  - Idade
  - CPF
  - Endereço
> Esses dados devem ser utilizados da classe Aluno

## Listagem dos alunos cadastrados
### O sistema deverá mostrar todos os alunos cadastrados
- A listagem deverá mostrar:
  - Alunos pré-cadastrados
  - Novos cadastros (não é necessário a persistência de dados)
- A listagem deverá conter os seguintes dados:
  - Nome
  - Idade
  - CPF
  - Endereço
 
## Alteração cadastral dos alunos
### O sistema deverá realizar a alteração cadastral dos alunos pré-cadastrados e recém cadastrados
- Através da alteração, será possível:
  - Alterar a Idade do Aluno
  - Alterar o Endereço do Aluno
  - Alterar o Nome do Aluno
> Acredito que o CPF não deveria ser alterável

## Exclusão de alunos
### O sistema deverá ser capaz de excluir um aluno ou mais
- A exclusão deverá deletar todos os dados daquele Aluno selecionado

## Registros na classe Aluno
### O sistema deverá ter uma classe Aluno para armazenar os campos:
- Nome
- Idade
- CPF
- Endereço
> Como segurança não será prioridade no momento, de início será public.
